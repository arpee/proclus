saito
Saito is a big-data blockchain. Try our demo:
  • Email (start here)
  • Facebook

Our goal is running uncensorable, free-speech web applications (Gmail, Google, Facebook, Twitter, etc.) on a terabyte-level blockchain. We achieve the scalability to do this by combining a transient blockchain with a new proof-of-transactions that pays nodes for contributing bandwidth to the network.

The above demos will run directly in your browser using javascript code that will be streamed to you from our server in Los Angeles. The download should be almost instantaneous in the United States, but can take up to a minute in China due to its size (~1.6 MB). Please be patient if you are trying Saito from a location with slower Internet access.
