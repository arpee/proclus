Saito is a big-data blockchain. Try our demos:
  • Search
  • Email
  • Secret

The above demos will run directly in your browser using javascript code that will be streamed to you from our server in Los Angeles. The download should be almost instantaneous in the United States, but can take up to a minute in China due to its size (~1.6 MB). Please be patient if you are trying Saito from a location with slower Internet access.
